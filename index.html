Here are 5 aspects/tools of Sass I used in this assigment

  1. I used $ to group css styles together
  2. I nested the child element p with the parent element header in sass
  3. inside the p element I did &:hover to do underline under "Sass assigment" in the header
  4. I used @mixin for header and footer so they would have all the same styling, and then @include in each of them. I over wrote the font size in footer
  5. I did a @media inside the header element
